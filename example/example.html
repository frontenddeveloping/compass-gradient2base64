Default usage example

Default using (without arguments) is equivalent to calling:
inline-gradient(linear, 100, 100, to left, #FFF 0%, #000 100%) 
                        some_css_selector {
                            background-image: inline-gradient();
                        }

Rainbow gradient example

Rainbow gradient from left to right.
Width is 150px and height is 500px.
This gradient is an image, it can't be with out fix size.
If you need to use the same gradient with other sizes try to use CSS rule background-size: 100% 100% 
                        some_css_selector {
                            background-image: inline-gradient(linear, 150, 500, to left, red 0%, orange 16.67%, yellow 33.34%, green 50%, lightskyblue 66.67%, blue 83.33%, violet 100%);
                        }
